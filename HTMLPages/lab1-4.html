Common HTML Tags

These zre the most commonly used HTML tags and a short description of each.

Block Elements

Block element take up their own line and take up the full width of the page.

Strong

This elemnt makes text bold

Inline Elements

Inline only takes up as much space on a line as they need to. They do not start new lines.

a

This tag defines a hyperlink.

Div

Defines a section a document

em

Defines emphasized text

body

Defines the document's body

main

Specifies the main content of a document

footer

Defines a footer for a document or section

li

Defines a list item

head

Defines information about the document

section

Defines a section of the document

Article

Defines a article

html

Defines the root of an HTML document

header

Defines a header for a document or a section

nav

Defines navigation links

aside

Defines content aside from the page content

img

Defines an img

span

Defines a section in a document
